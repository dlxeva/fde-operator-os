# Product Feedback

## Delivery Lessons Worth Productization

- Coordinators need an overdue-resolution view because the system of record does not make exception ownership visible.
- Closure evidence validation should be a reusable policy module, not custom logic per deployment.
- Duplicate-exception linking is a candidate backlog item because it affects both trust and reporting quality.

## Internal Tooling Implications

- A reusable evidence-check rule pack would reduce per-project redefinition.
- A starter eval pack for exception-closure loops should ship with disputed and low-confidence cases.
