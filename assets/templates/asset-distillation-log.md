# Asset Distillation Log

## Use

Use after a pilot or delivery loop has produced enough evidence to promote reusable assets.

## Required Fields

- Source case
- Asset candidate
- Asset type
- Reuse target
- Source evidence
- Promotion decision
- Owner
- Next maintenance action

## Promotion Standard

An entry is complete only if another operator could reuse the asset without needing the original project operator to explain it live.
